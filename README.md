Этот репозиторий создан для обучения работе с Git и GitHub, а также для систематизации учебных материалов, расчетов, чертежей и отчетов по курсовым проектам. Здесь будет храниться история всех этапов моей работы.
Конспекты лекций, лабораторные работы по CAD-программам (AutoCAD, Revit), расчеты в MathCAD и Excel (по механике грунтов, железобетонным конструкциям), эскизы и чертежи объектов, отчеты по практикам и курсовым проектам, полезные скрипты для автоматизации расчетов.
